크림슨우드 내비게이션 온라인 배포 메모

1. 이 폴더 전체를 Cloudflare Pages, GitHub Pages 등의 정적 사이트 호스팅에 올립니다.
2. index.html이 사이트 최상위에 있어야 합니다.
3. 독립 도메인을 연결한 뒤 HTTPS 접속과 모바일 화면을 확인합니다.
4. 이 버전에는 광고, 분석, 추적, 외부 스크립트가 없습니다.
5. 실제 운영 전 운영자 연락처와 저작권/자료 사용 권한을 확인합니다.

현재 사이트는 서버 DB 없이 브라우저에서 경로를 계산합니다.
마지막 출발지·도착지·선택 경로는 각 방문자의 브라우저 localStorage에만 저장됩니다.
사이트는 외부 주소로 데이터를 전송하지 않으며, _headers에 보안 헤더가 설정되어 있습니다.
호스팅 계정에는 반드시 긴 고유 비밀번호와 2단계 인증을 사용하세요.
