크림슨우드 내비게이션
========================

홈페이지
https://muti-xd.github.io/crimsonwood-navi-web/

사용 방법
1. 위 홈페이지에 접속합니다.
2. 출발지와 도착지를 선택합니다.
3. 두 지점이 모두 선택되면 최단 경로가 즉시 계산됩니다.

주요 기능
- 출발지·도착지 선택 시 최단 경로 자동 검색
- 여러 최단 경로 탭 전환
- 스킬 필요 구간 표시
- 지도 이미지 확대
- 즐겨찾기 및 마지막 검색 저장
- PC/모바일 반응형 화면
- 모든 경로 계산을 브라우저 안에서 처리

안내
- 별도의 프로그램 설치가 필요하지 않습니다.
- 브라우저 데이터를 삭제하면 즐겨찾기와 마지막 검색 기록도 초기화됩니다.
